Como instalar

Paso 1

Descargar el archivo

recibiras un link, con el mismo vas hacer la descarga en tu celular de un archivo comprimido con nombre Retro-Game.Rar. Se recomienda hacer un respaldo de su archivo en un pendrive, en su computadora o cualquier otro medio para una futura instalación en otro dispositivo.

Paso 2

Descomprimir archivo

Para descomprimir el archivo recomendamos usar el aplicativo ZArchiver, abre la app ZArchiver y navega hasta la carpeta Download donde encontraras el archivo Retro-Game.rar, luego descomprimir para el directorio raiz, el proceso puede tardar algunos minutos.

Descargar ZArchiver
Paso 3

Instalar sistemas

Despues de descomprimir las carpetas 2-Jogos y Apps_backup_reinstall, el siguiente paso es instalar el sistema Retro Game con la app Apps Backup Restore
