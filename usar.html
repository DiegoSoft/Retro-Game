Como usar

Como salir del juego pantalla horizontal

Para salir del juego y volver para la lista debes mantener precionado el boton Select y luego preciona el boton Star

Como salir del juego pantalla vertical

Para salir del juego y volver para la lista debes mantener precionado el boton Select y luego preciona el boton Star

Como volver al menu principal

Para volver al menu principal desde la lista de juegos, solo debes precionar el boton de volver o regresar de tu celular.

Como agregar juegos a favoritos

Para favoritar juegos debes hacerlo desde la lista, presiona por 3 segundos el juego que quieras, luego en el sub menu buscar la opción (Add favorites).
